Channels

Manage your video channels. Each channel can have multiple episodes, characters, and visual styles.

GET /channels

List all channels for the authenticated user.

Response

{
  "channels": [
    {
      "id": "clx...",
      "name": "My Channel",
      "niche": "Educational content about space",
      "characters": [...],
      "_count": { "episodes": 10, "characters": 3 }
    }
  ],
  "meta": { "total": 1 }
}
POST /channels

Create a new channel.

Request Body

ParameterTypeDescription
name requiredstringChannel name
niche requiredstringChannel niche/topic description
description optionalstringDetailed channel description
dialogueLanguage optionalstring"vi" or "en" (default: "vi")
GET /channels/{id}

Get channel details including all episodes and characters.

Episodes

Create and manage video episodes. Each episode contains multiple scenes with AI-generated prompts for video generation tools like Veo or Sora.

GET /channels/{id}/episodes

List all episodes for a channel with their scenes.

POST /channels/{id}/episodes

Create a new episode with AI-generated content. The AI will generate a complete video script with scenes, prompts, and optional YouTube metadata.

Request Body

ParameterTypeDescription
topicIdea requiredstringMain topic or title for the episode (e.g., "The Mystery of Black Holes")
customContent optionalstringPre-scraped content to base the episode on (e.g., article text)
totalScenes optionalnumberNumber of scenes to generate (default: 10, range: 5-20)
cinematicStyle optionalstringVisual style: "cinematic_documentary", "psychological_drama", "sitcom_comedy", etc.
voiceOverMode optionalstring"with_host" | "narrator_only" | "no_voice"
voiceGender optionalstring"auto" | "male" | "female"
voiceTone optionalstring"warm" | "energetic" | "calm" | "mysterious" | "professional"
useCharacters optionalbooleanInclude channel's characters (default: true)
selectedCharacterIds optionalstring[]Specific character IDs to include
categoryId optionalstringCategory ID for episode organization
generateYoutubeStrategies optionalbooleanGenerate YouTube titles, description, tags (default: true)
narrativeTemplateId optionalstring"storytelling" | "documentary" | "tutorial" | "review" | "vlog"

Response

{
  "success": true,
  "episode": {
    "id": "clx...",
    "title": "The Fascinating Mystery of Black Holes",
    "synopsis": "Explore the cosmic phenomena...",
    "scenesCount": 10,
    "scenes": [
      {
        "order": 1,
        "title": "Opening Hook",
        "promptText": "[VOICEOVER: Deep, mysterious tone] What happens when you fall into a black hole? ENVIRONMENT: Vast space with swirling galaxy...",
        "voiceover": "What happens when you cross the event horizon...",
        "duration": 8
      }
    ],
    "youtubeStrategies": {
      "titles": ["What Really Happens Inside a Black Hole?", ...],
      "description": "In this video, we explore...",
      "tags": ["black hole", "space", "physics", ...]
    }
  }
}
DELETE /channels/{id}/episodes?episodeId={episodeId}

Integration Examples

cURL

# Create an episode
curl -X POST https://your-domain.com/api/v1/channels/YOUR_CHANNEL_ID/episodes \
  -H "Content-Type: application/json" \
  -H "x-api-key: veo_your_api_key" \
  -d '{
    "topicIdea": "10 Amazing Facts About the Ocean",
    "totalScenes": 12,
    "cinematicStyle": "cinematic_documentary",
    "voiceTone": "energetic"
  }'

Make.com / Zapier HTTP Module

Method: POST
URL: https://your-domain.com/api/v1/channels/{channelId}/episodes

Headers:
  Content-Type: application/json
  x-api-key: {{your_api_key}}

Body (JSON):
{
  "topicIdea": "{{scraped_title}}",
  "customContent": "{{scraped_content}}",
  "totalScenes": 10,
  "generateYoutubeStrategies": true
}

Python

import requests

response = requests.post(
    "https://your-domain.com/api/v1/channels/YOUR_CHANNEL_ID/episodes",
    headers={
        "Content-Type": "application/json",
        "x-api-key": "veo_your_api_key"
    },
    json={
        "topicIdea": "The Science of Dreams",
        "totalScenes": 10,
        "cinematicStyle": "psychological_drama",
        "voiceTone": "mysterious"
    }
)

episode = response.json()
print(f"Created episode: {episode['episode']['title']}")

JavaScript / Node.js

const response = await fetch(
  'https://your-domain.com/api/v1/channels/YOUR_CHANNEL_ID/episodes',
  {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'x-api-key': 'veo_your_api_key'
    },
    body: JSON.stringify({
      topicIdea: 'Secrets of Ancient Civilizations',
      totalScenes: 15,
      cinematicStyle: 'cinematic_documentary',
      narrativeTemplateId: 'documentary'
    })
  }
);

const { episode } = await response.json();
console.log('Created:', episode.title);
